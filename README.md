<h3> How can you run it?</h3>

Clone the project

```bash
  git clone https://github.com/farildev/spotify-clone.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build the project

```bash
  npm run build
```
